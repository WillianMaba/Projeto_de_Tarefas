# 📝 Gerenciador de Tarefas (CLI) — Python + SQLite

## 📌 Descrição

Sistema de gerenciamento de tarefas em linha de comando (CLI), desenvolvido em Python, com persistência de dados utilizando SQLite.

O projeto foi construído com foco em práticas iniciais de desenvolvimento back-end, incluindo:
- Organização modular
- Manipulação de banco de dados
- Implementação de operações CRUD

---

## 🎯 Objetivo

Este projeto foi desenvolvido com fins acadêmicos e tem como principais objetivos:

- Praticar lógica de programação  
- Aplicar conceitos de CRUD  
- Trabalhar com banco de dados SQLite  
- Organizar um projeto em camadas simples  
- Simular um sistema real de gerenciamento  

---

## 🚀 Funcionalidades

- ✅ Criar tarefas  
- 📋 Listar tarefas cadastradas  
- ✏️ Atualizar tarefas existentes  
- ❌ Excluir tarefas  
- 💾 Persistência de dados em SQLite  

---

## 🛠️ Tecnologias Utilizadas

- Python 3  
- SQLite3  
- Estrutura modular com múltiplos arquivos  

---

## 📂 Estrutura do Projeto

```bash
Projeto_de_Tarefas/
│
├── main.py                # Interface principal (menu CLI)
├── facilitadores.py      # Funções auxiliares
│
└── database/
    ├── __init__.py
    ├── conexao.py        # Conexão com o banco
    └── operacoes.py      # CRUD de tarefas
```
